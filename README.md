# Getting Started

## Prerequisite
You must have these installed and working correctly. You can google for installation.
1. NodeJs
2. MongoDB

### API
```bash
# Go inside the directory
cd api

# Install dependencies for api
npm install

# Start API
npm start
```

### Chat Server
```bash
# Go inside the directory
cd server

# Install dependencies for api
npm install

# Start API
npm start
```

### Chat Client
```bash
# In main directory, install dependencies for api
npm install

# Start API
npm start
```
